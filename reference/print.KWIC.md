# Prints summary of `KWIC` object.

`print.KWIC` prints to console the summary of an object of class `KWIC`
including the database fields(columns) used, the total number of
keywords and the number of distinct keywords in the index.

## Usage

``` r
# S3 method for class 'KWIC'
print(x, ...)
```

## Arguments

- x:

  An object of class `KWIC`.

- ...:

  Unused
